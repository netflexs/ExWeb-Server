
# ExWebServer

A WebServer Made with ExpressJS and PHP




## Installation

Install Package and Run

```bash
  npm install
  node .
```
    
## Feature

- Additional PHP Support using PHP Path

- Anti Flooding

- Rate Limiter

- Network Throttling

- HTTP Logger (All http Methode and Byte Logger)
